## meteor-python
Launch Python commands as worker threads from Meteor.  This implementation
bypasses the command shell, and uses process-to-process intercoms.    

=============================================
#### Installation   

```sh
git clone http://github.com/awatson1978/meteor-python  
cd meteor-python  
meteor  
```  

=============================================
